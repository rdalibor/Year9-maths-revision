Distance-Time Graphs

Key Concepts

Horizontal lines: The object is stationary (stopped).

Diagonal lines: The object is moving at a constant speed.

Gradient: Represents the speed. Steeper lines = Faster speeds.

Speed = Distance ÷ Time

Example Journey

A 30km trip, a 1-hour rest, then a 30km return trip.
